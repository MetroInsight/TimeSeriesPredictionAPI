get /models/{uuid}/anomalies
Get anomalies from a specific model (modelsUuidAnomaliesGet)
Returns anomalies detected by the given model. Anomalies are those points in the target time series for which the prediction error is highest. Any error value larger than 3 standard deviations is considered an anomaly, and anomalies are returned in sorted order by prediction error.

Path parameters

uuid (required)
Path Parameter — UUID of the target time series

Query parameters

count (optional)
Query Parameter — Maximum number of anomalies to return

Return type

anomalies

Example data

Content-Type: application/json
{
  "time" : [ 123456789 ],
  "error" : [ 1.3579000000000001069366817318950779736042022705078125 ]
}

get /models/{uuid}/correlations
Get correlations from a specific model (modelsUuidCorrelationsGet)
Get features which have the highest correlation with the target time series, determined by measuring the increase in prediction error when the feature is removed.

Path parameters

uuid (required)
Path Parameter — UUID of the target time series

Query parameters

count (optional)
Query Parameter — Maximum number of correlations to return default: 20

get /models/{uuid}/forecast
Get forecasts from a specific model (modelsUuidForecastGet)
Returns future values predicted by the model for the last observed sensor values. The number of forecast depends on the number of output values specified when the model was created. Return type is a list of anomalies, where each anomaly is a list containing two items: the time and the forecasted sensor value.

Path parameters

uuid (required)
Path Parameter — UUID of the target time series

Return type
